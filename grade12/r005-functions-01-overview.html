Quadratics

 

 

The Parabola

Three forms of a parabola

 

y = a (x-p)2 + q vertex form
y = a x2 + bx + c standard form
y = a (x- r1) (x-r2) intercept form

 

 

a Determines the shape

 

if a > 0 Parabola is concave UP - smile

 

if a < 0 Parabola is concave DOWN – grim

 

The further a from 0 the thinner the graph

 

Vertical Shift

 

The greater the value of q or c the greater

the vertical shift.

 

Positive values of q or c

move the graph UP,

 

Negative values of q or c

move the graph DOWN

 

 

 

Axis of symmetry

 

A Parabola is always vertically symmetrical.

The axis of symmetry is half way

between the Roots (where y = 0)

 

The Turning Point

 

The turning point is on

the Axis of Symmetry.

In vertex form this point is (p,q)

 

Hyperbola

 

 

The Hyperbola

The Hyperbola

 

 

\(y = {\color{red}{ a} \over x - \color{green}{ p}} + \color{blue}{ q} \)

 

 

a determines shape

 

q determines vertical shift

 

p determines horizontal shift

 

Shape

 

a > 0

 

Hyperbole lies in 1st and 3rd quadrant

 

 

 

a < 0

 

 

Hyperbole lies in 2nd and 4th quadrant

 

 

Axis of Symmetry

 

Hyperbole is Symmetrical about :

 

y = x - p + q and y = x + p + q

 

 

 

Domain and Range

 

 

Domain

 

x ε ℝ or x ε (- ∞ ; ∞) and x ≠ p

 

Range

 

 

y ε ℝ or y ε (- ∞ ; ∞) and y ≠ q

 

 

Intersection with Axis of Symmetry

 

a > 0

 

 

Intersects y = x - p + q at

 

 

(√a + p ; √a + q)

 

 

and

 

(-√a + p ; -√a + q)

 

 

 

 

a < 0

 

Intersects y = x - p + q at

 

 

(-√a + p ; √a + q)

 

and

 

(-√a + p ; -√a + q)
